What's New? - Version {{ version }} Changelog

Version

New Features

  • Decks: You can now create and view mech decks! Head to the new Deck Library screen to view existing decks and create new ones. From there you can start adding cards to make up your deck and view them all in one place!

New Features

  • Card Manufacturers: Added a new "Manufacturer" field to cards that allows you to select a part manufacturer from a dropdown.
  • Enabled Core Stats on Hardpoints and Mods: The armor, agility, and energy stat inputs will now appear for Hardpoints and Mods on the card create page so those parts can now grant bonuses or banes to those stats.
  • Ability Text Keyword Descriptions: Descriptions of card keywords are now shown when hovering over a highlighted keyword in a card's ability text.

Bug Fixes

  • Ability Text Keyword Bug: Fixed an issue on the card create screen where only the first instance of a named keyword would be highlighted. All named keywords in a card's ability text will now be properly highlighted.

New Features

  • Card Activation Icons: Added additional icon parsing to ability text. You can now use "Shoot", "Melee", or "Activate" as icon keywords.

New Features

  • Card Version Editing: Added the ability to edit previous versions of a card on the card edit screen. Past versions can now be tweaked or corrected if needed and can also be saved as new versions.
  • Card Deletion: Added the ability to delete all versions of a card, or the most recent version of a card from the card edit screen.
  • Pending Cards: Newly created cards are now added in a pending state while being worked on ahead of approval. You can search through pending cards on the "Pending Cards" screen. Cards that have not yet been approved may be approved by clicking the "approve" button on the card view screen.

Quality of Life Improvements

  • Card Version Sort Order: All dropdowns showing card versions will now properly show in numerical order.
  • Card Search Sort Order: Cards shown when searching cards in the card library are now sorted by type, then by name.
  • Card Search Retention: Filter criteria is now saved when navigating back to the card library page.

Bug Fixes

  • Card View Details Cutoff: Fixed an issue on the card view screen where details on some cards with longer ability text were getting cut off by the buttons at the bottom of the pane.
